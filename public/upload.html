Upload Instructions

Upload Excel files (.xlsx) containing your network data. The system supports the following data types:

  • Nodes: Network locations (depots, customers, metro stations)
  • Vehicles: Available vehicles (trucks, drones) with specifications
  • Demands: Delivery requests with origins, destinations, and quantities
  • Metro Timetable: Public transport schedules
  • Parameters: System configuration settings
